# Representative outputs

This folder contains representative outputs from the Aspen Plus runs to enable verification.

## Recommended files (examples)
- `Mass_Energy_Balance.xlsx` : Overall mass and energy balance summary
- `Key_Performance_Metrics.csv` : Conversion, selectivity, yields, utility demands
- `Stream_Tables.xlsx` : Key stream results exported from Aspen (optional)

## Notes
These outputs are provided so that reviewers/readers can verify the reported results
even if they do not have access to Aspen Plus.
