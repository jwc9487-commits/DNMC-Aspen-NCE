# DNMC Aspen Plus Process Model (Joule submission)

This repository is **publicly accessible** and does **not require any login or personal information**.

It provides the Aspen Plus process model, algorithms, input data, and representative outputs used in the manuscript to ensure transparency and reproducibility.

## Repository structure
- `Aspen_Model/` : Aspen Plus model files (e.g., `.bkp`)
- `Model_Description/` : Model structure, assumptions, and algorithms
- `Input_Data/` : Input parameters and operating conditions
- `Output_Results/` : Representative outputs for verification
- `Reproducibility/` : Step-by-step reproduction guide

## How to reproduce
See: `Reproducibility/Step_by_Step_Reproduction.md`

## Notes on software access
Aspen Plus is commercial software; execution of the model requires a valid Aspen license.
However, the full model structure, assumptions, input data, and representative outputs are provided here
to allow reviewers/readers to access and evaluate the model without logging in or providing personal information.

## Citation
If you use this repository, please cite the associated Joule manuscript.
